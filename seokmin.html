2. Creating an RC CAR that follows certain object we wants.

I carried out the project using RC cars made by combining microcontrollers, motors, and various sensors. In this project, the RC car detects people on the screen, selects one person that the user wants, and follows only that person. About 30 pages of this project report that I wrote myself are kept in the lab material. This project use not only object detecting but object tracking using byte tracker.


3. Creating a Drone that follows and attacks certain object we want.

When I worked on the project with the RC car, I could only follow the object at a slow speed due to the hardware's response speed limit. However, by using the Telo Edu drone with strong hardware, it is possible to follow the desired object well at high speed. In this process, control algorithms such as PID control were used, and the concept of signal was also used to freely change modes such as keyboard mode, tracking mode, and attacking mode. Even when there is more than one person on the screen, only the set target follows well.
